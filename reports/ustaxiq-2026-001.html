Core Web Vitals Signals — Score: 30/100
CriticalLCP: Hero image is a CSS background — invisible to preload scanner
The largest visible element on load (hero image) is a CSS background-image, not an <img> tag. Browsers cannot preload it. This means the Largest Contentful Paint is severely delayed — Google penalizes anything over 2.5 seconds.
Replace hero CSS background with an <img> tag using fetchpriority="high" and loading="eager".
CriticalZero images use loading="lazy"
All 10 images on the page load eagerly, including images below the fold. This wastes bandwidth, delays LCP, and degrades mobile performance significantly.
Add loading="lazy" to all images below the fold. Keep fetchpriority="high" only on the hero.
HighNo font-display strategy — invisible text on load (FOIT)
Google Fonts (Montserrat) and the custom icon font load without font-display: swap. Users see blank space where text should be while fonts download.
Add &display=swap to Google Fonts URL. Add font-display: swap to @font-face declarations.
HighYouTube thumbnail image missing width and height attributes
An image loaded from i.ytimg.com has no width/height declared. When it loads, it pushes content down — causing Cumulative Layout Shift (CLS), a Core Web Vitals failure.
Add explicit width="" height="" to all img tags. Reserve space before content loads.
HighCallRail tracking script blocks rendering
The CallRail phone number swap script (cdn.callrail.com) loads in the document head with no defer attribute. It is a third-party script that holds up the entire page render.
Add defer to the CallRail script tag. CallRail's async/defer mode is fully supported.
Indexability Issues — Score: 60/100
Critical10+ utility/test/template pages are indexed in sitemap
Google is crawling and indexing: /logout/, /login/, /test-book-consultation/, /test-book-consultation-main/, /events-template/, /venue-template/, /taxonomy-template/, /toggle-bar/, and duplicate /thank-you/ + /thankyou/ pages. These dilute site quality.
Set noindex + remove from sitemap in Yoast. Delete or 301-redirect duplicate thank-you pages.
MediumDuplicate webinar pages consuming crawl budget
Both /member-area-early-planning-2018-tax-year/ and /early-planning-2018-tax-year/ appear to be near-duplicate versions of the same stale 2018 content from 6+ years ago.
301-redirect member-area versions to public versions. Update or consolidate stale content.
MediumWordPress version publicly exposed in <head>
<meta name="generator" content="WordPress 6.9.4"/> tells hackers exactly what to exploit. Yoast and Hummingbird version numbers also exposed in HTML comments.
Add remove_action('wp_head', 'wp_generator') to functions.php. Disable version comments in Hummingbird and Yoast settings.
MediumNo hreflang tags for bilingual/bicountry audience
The site serves both Canadian and US audiences but has no hreflang tags to signal geographic targeting to Google. og:locale is set to en_US but HTML lang is en-CA — inconsistent.
Add hreflang="en-ca" and hreflang="en-us" alternate tags. Fix og:locale to en_CA.
LowIndexNow protocol not implemented
Yoast SEO Premium includes IndexNow support. When new content is published, Google/Bing are not immediately notified, causing indexing delays of hours or days.
Enable IndexNow in Yoast SEO → Integrations if on Premium. Otherwise install Bing URL Submissions plugin.
03 — On-Page SEO

Meta Tags, Headings & Structured Data

Analysis of title tags, meta descriptions, heading hierarchy, internal linking, and schema markup across all 41 pages.
Meta Tag Audit
CriticalMeta descriptions missing on 10+ key pages
No meta description on: About Us, Contact Us, all 6 service pages, Blog page, FAQ page, Our Process page, and Careers. Missing meta descriptions result in Google auto-generating snippets — often poorly — reducing click-through rate from search results by an estimated 3–7%.
Write unique 150–160 character meta descriptions for all pages. Include primary keyword + clear value proposition + call to action.
HighPage titles are generic — missing keywords and value
Titles like "About Us", "Contact Us", "Our Process" appear without the brand name, target keywords, or location. Compare to competitor titles: "Cross-Border Tax Services for Canadians | US Tax IQ | Oakville ON".
Rewrite all page titles: [Primary Keyword] — [Location/Audience] | U.S. Tax IQ. Max 60 characters.
CriticalNo LocalBusiness / AccountingService schema markup
Two physical offices, an IRS certification, and specialized credentials — but zero LocalBusiness structured data. Google cannot display business details in the Knowledge Panel or Local Pack without this schema.
Add AccountingService JSON-LD with: name, address (both locations), telephone, openingHours, areaServed, aggregateRating.
HighFAQ schema (microdata) may conflict with Yoast JSON-LD
Homepage FAQ uses inline schema.org microdata, while Yoast generates JSON-LD for other types. Having two competing schema formats on the same page confuses parsers and may invalidate rich result eligibility.
Convert FAQ microdata to JSON-LD format consistent with Yoast output, or use Yoast's built-in FAQ block.
MediumImages have generic filenames (Untitled-design-XX.png)
Multiple images are served with filenames like Untitled-design-28.png, Untitled-design-84.png — providing no keyword context for Google Image Search.
Rename all image files with descriptive slugs before uploading. Update alt text to match.
Medium3 images missing alt text entirely
3 of 10 <img> tags on the homepage have no alt attribute — an accessibility violation (WCAG 2.1) and a missed SEO signal.
Add descriptive alt text to every image. Decorative images use alt="".

Add an inline CTA block after every 3rd paragraph and at article end. Offer a free "US Tax for Canadians" checklist PDF in exchange for email.

Research consistently shows that placing star ratings and review counts adjacent to CTA buttons increases conversions by 15–30%. The 5-star rating is mentioned once on the homepage — not positioned near any booking button.
Add "★★★★★ Trusted by 100+ clients" directly under every CTA button across all pages.

Not every visitor is ready to book a paid consultation. Without a free resource (checklist, guide, email course), there is no way to capture and nurture the 80–90% who aren't ready to buy immediately.
Create 2–3 downloadable resources: "Cross-Border Tax Checklist for Canadians", "FBAR Filing Guide", "Expat Tax Deadline Calendar". Gate with email.
